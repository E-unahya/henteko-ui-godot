# 今後の予定

今後作成する予定のUIパーツはこちら

- 勝手に踊りだすテキスト →　読めない！！！ ←　一応実装完了
- こんな名前入力はやだ　→　入力している最中に４ぬ
- 変なオプションボタン　→　落ちたらそのままだらり
- 変なスクロールバー →　スクロールしようとしたら違う方向へ行く
- 変なテキストフィールド　→　おかしい・・・ちゃんと入力したはずなのに・・・
